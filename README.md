# Awesome Hyvä Resources for Magento 2
A curated, up-to-date list of free and paid resources for Hyvä theme development (2024–2025).  
Note: As of Nov 2025, the Hyvä Theme core is fully free and open source (OSL3 + AFL3).

---

## 1. Official Hyvä Resources

- https://docs.hyva.io/  
  Official Hyvä documentation.

- https://hyva.io/blog.html  
  Release notes and announcements.

- https://gitlab.hyva.io/  
  Hyvä GitLab (theme is free; other products remain commercial).

- https://gitlab.hyva.io/hyva-public/module-tracker  
  Third-party module compatibility tracker.

- https://docs.hyva.io/welcome/slack-channels.html  
  Official Slack channels.

- https://docs.hyva.io/hyva-themes/magewire/introduction.html  
  MageWire reactive layer.

- https://docs.hyva.io/hyva-admin/introduction.html  
  Hyvä Admin (free backend UI framework).
