# Awesome Hyvä Resources for Magento 2

[**Hyvä**](https://www.hyva.io/) (pronounced **Hoo-vuh** — derived from the Finnish word for "good") is a modern, high-performance frontend for Magento 2. It replaces the complex, legacy tech stack (Knockout.js, RequireJS, Less) with a streamlined architecture based on **Tailwind CSS** and **Alpine.js**.

A curated list of free and paid resources for Hyvä theme development (2024–2026).

**Current Status:**
As of November 2025, the Hyvä Theme core is open source (OSL-3.0 and AFL-3.0 licenses). It is a widely adopted frontend for Magento 2 and Adobe Commerce, utilized by numerous agencies for its performance benefits and developer experience.

## 1. Official Hyvä Resources
- [Official Hyvä Documentation](https://docs.hyva.io/) – Comprehensive technical guides and the primary starting point for developers.
- [Official Hyvä Blog](https://www.hyva.io/blog) – Release notes, case studies, and roadmap announcements.
- [Hyvä GitLab](https://gitlab.hyva.io/) – Hosting for commercial products and contribution area for partners.
- [Official Hyvä Themes GitHub](https://github.com/hyva-themes) – Organization for open-source core modules and community contributions.
- [Default Hyvä Theme Repository](https://github.com/hyva-themes/magento2-default-theme) – The base theme package for Magento 2 storefronts.
- [Hyvä Theme Module](https://github.com/hyva-themes/magento2-theme-module) – The core framework module providing the infrastructure for Tailwind and Alpine.js integration.
- [Compatibility Module Tracker](https://gitlab.hyva.io/hyva-public/module-tracker) – Community-maintained list of third-party module compatibility status.
- [Hyvä Slack Community](https://docs.hyva.io/welcome/slack-channels.html) – Active developer support and discussion channels.
- [MageWire Documentation](https://docs.hyva.io/hyva-checkout/magewire/index.html) – Guide for the reactive layer used in Hyvä Checkout and advanced components.
- [Hyvä Admin](https://docs.hyva.io/hyva-admin/index.html) – Modern backend UI framework; alternative documentation available at [hyva-themes.github.io](https://hyva-themes.github.io/magento2-hyva-admin/).
- [Hyvä x-snap-slider](https://docs.hyva.io/hyva-themes/working-with-alpinejs/alpine-plugins/x-snap-slider.html) – Native Alpine.js-based slider implementation.
