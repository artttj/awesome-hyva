# Awesome Hyvä Resources for Magento 2

A curated, up-to-date list of free and paid resources for Hyvä theme development (2024–2026).

**Note:** As of November 2025, the Hyvä Theme core is fully free and open source (OSL3 + AFL3) and available on GitHub.

---

## 1. Official Hyvä Resources
- https://docs.hyva.io/ Official Hyvä documentation.
- https://www.hyva.io/blog Official blog (release notes and announcements).
- https://gitlab.hyva.io/ Hyvä GitLab (commercial products remain commercial; core theme now on GitHub).
- https://github.com/hyva-themes Official Hyvä Themes GitHub organization.
- https://github.com/hyva-themes/magento2-default-theme Official default Hyvä Theme repository (free core).
- https://github.com/hyva-themes/magento2-theme-module Official core framework module.
- https://gitlab.hyva.io/hyva-public/module-tracker Third-party module compatibility tracker.
- https://docs.hyva.io/welcome/slack-channels.html Official Slack channels.
- https://docs.hyva.io/hyva-checkout/magewire/index.html MageWire reactive layer (now documented under Hyvä Checkout).
- https://docs.hyva.io/hyva-admin/index.html Hyvä Admin (free backend UI framework; full docs at https://hyva-themes.github.io/magento2-hyva-admin/).
- https://docs.hyva.io/hyva-themes/working-with-alpinejs/alpine-plugins/x-snap-slider.html Hyvä x-snap-slider.
